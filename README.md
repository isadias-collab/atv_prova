# Atividade 🌎 ChileVenture

Projeto de landing page para uma agência de viagens focada no Chile, apresentando destinos turísticos, pacotes, atividades e ofertas especiais.

---

## 📌 Sobre o Projeto

O **ChileVenture** é uma página web estática desenvolvida com **HTML e CSS**, com o objetivo de simular um site moderno de turismo.  

A interface apresenta:

- Banner principal com chamada de impacto
- Barra de navegação
- Sistema de busca de viagens
- Destinos em destaque
- Pacotes populares
- Atividades turísticas
- Ofertas especiais
- Galeria de fotos
- Rodapé completo

---
